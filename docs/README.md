# Documentation

This documentation is organized with the Diataxis framework:

- **Tutorials** guide you through a successful learning path.
- **How-to guides** help you complete a specific task.
- **Reference** gives exact facts about scripts, templates, commands, and configuration.
- **Explanation** describes why the system is shaped the way it is.

## New to This Repo

Start with [Overview](overview.md). It explains what this repository does, who it is for, and which path to take next.

If your goal is to build a working machine, go directly to [First Workstation](tutorials/first-workstation.md).

If you are changing docs, read [Contributing Documentation](contributing-docs.md).
