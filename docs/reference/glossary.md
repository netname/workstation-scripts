# Glossary

**Age**: A small encryption tool used with SOPS. The private key must be backed up separately.

**Devenv**: Project-local environment manager used to declare runtimes, language servers, formatters, and other project tools.

**Direnv**: Shell hook that activates a project environment when entering a directory.

**Dotfiles**: The private repository that contains personal workstation configuration.

**Home Manager**: Nix-based user environment manager for packages, shell config, git config, and generated dotfiles.

**Nix**: Reproducible package manager that installs tools into immutable store paths.

**SOPS**: Tool for encrypting secrets that can be committed safely.

**Sessionizer**: Script that finds project directories and opens or attaches tmux sessions.

**tmux**: Terminal multiplexer used for persistent shell workspaces.

**Two-repository model**: Pattern where public bootstrap scripts live in `workstation-scripts` and private personal configuration lives in `dotfiles`.

**User-managed dotfile**: A file edited directly in dotfiles and reloaded by the owning tool instead of regenerated by Home Manager.

**WezTerm**: Terminal emulator used by the optional graphical desktop setup.
